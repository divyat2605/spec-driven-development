# AgentClinic Roadmap

## Phase 1: Core agent and ailment foundation
- initialize a TypeScript-based Next.js project
- define the data model for agents, ailments, and therapy types
- build CRUD screens for agents and ailments
- verify basic browser compatibility and responsive layout

## Phase 2: Therapy management and appointments
- add therapy objects and treatment plans
- implement appointment booking for agents and staff
- create booking validation and simple availability rules
- add a basic therapy detail page

## Phase 3: Dashboard and staff workflows
- build staff dashboard views for agents, bookings, and therapies
- add search, filtering, and navigation for clinic workflows
- introduce agent detail pages with ailment history and treatment notes
- add staff actions for booking management and therapy assignment

## Phase 4: Polish and marketing-ready UI
- improve visuals, spacing, and browser polish for modern UX
- add brand styling, hero content, and marketing-friendly copy
- implement accessibility checks and cross-browser testing
- prepare for deployment with CI, linting, and test automation

## Minimal phase structure
1. set up project + typed models
2. agent/ailment CRUD
3. therapy scheduling + booking
4. staff dashboard + support flows
5. design polish + launch readiness
