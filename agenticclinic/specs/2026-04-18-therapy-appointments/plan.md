# Plan for Therapy Management and Appointments

1. Therapy Types management
   - build CRUD API routes for therapy types
   - build list page showing all available therapy types
   - build detail page for a single therapy type
   - allow staff to create and delete therapy types

2. Appointment booking
   - build CRUD API routes for appointments
   - build appointments list page showing all bookings
   - build booking form to assign an agent to a therapy type with date and time
   - allow staff to cancel and complete appointments

3. Agent detail enhancement
   - add appointments section to the agent detail page
   - show upcoming and past appointments per agent

4. Validation and review
   - test all API routes for therapy types and appointments
   - verify booking flow works end to end in the browser
   - review against mission goals and roadmap before merging
